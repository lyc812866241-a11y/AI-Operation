# Bootstrap Corrections Log

> 记录每次 `[初始化项目]` 过程中 AI 的遗漏、错误判断和用户纠正。
> **下次初始化时，Phase 2 开始前必须先读此文件。**

---

## 使用规则

- 每次 Phase 3 校准时，如果用户指出草稿有遗漏或错误，AI **必须**在此记录，不可跳过
- 格式严格按下方模板填写
- **同一类错误出现 3 次** → 升级为 Phase 2 的强制扫描步骤（需同步更新 `skills/project-bootstrap/SKILL.md`）
- **30 天内未再出现的错误** → 移入 `## 已解决` 区块
