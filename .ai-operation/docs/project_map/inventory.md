# Project Inventory (资产清单)

> **[用途说明]**
> 本文件记录项目的**可枚举资产**——模块清单、skill 清单、API 清单等列表型数据。
> 这些数据不适合放在 activeContext.md（会被压缩）或 systemPatterns.md（格式不同）。
>
> **本文件的特殊规则：**
> - 每次更新必须是**完整列表覆写**，不是增量追加
> - AI 更新前必须先读取当前列表，合并后写入完整版本
> - 数量必须与实际一致——如果实际有 40 个 skill，文件里必须列 40 个
>
> **格式：每个资产类别一个 section，内容为 Markdown 表格或列表**
